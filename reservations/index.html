Stworzenie nowej rezerwacji

POST /reservations

Parametry

seats
  • ilość miejsc do rezerwacji
date
  • data w formacie yyyy-mm-dd
hour
  • godzina rezerwacji
notes
  • informacja do przekazania do restauracji
restaurant_id
  • id restauracji, w której powinna zostać złożona rezerwacja

Przykładowe parametry

{
  "seats": 2,
  "date": "2012-02-20",
  "hour": "15:30",
  "notes": "Poprosze o stolik przy oknie",
  "restaurant_id": 118
}

Odpowiedź

Poprawne zapytanie zwróci dane rezerwacji:

Status: 200 OK
{
  "id": 18,
  "notes": "Poprosze o stolik przy oknie",
  "restaurant_id": 118,
  "seats": 2,
  "date": "2012-02-20",
  "hour": "15:30"
}

”
