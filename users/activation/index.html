Rejestracja użytkowników

POST /activations

Parametry

activation_token
  • kod aktywacyjny, który użytkownik dostanie w smsie

Odpowiedź

Poprawne zapytanie zwróci aktywowanego użytkownika:

Status: 200 OK
{
  "first_name": "Jan",
  "last_name": "Kowalski",
  "phone": "344-555-111",
  "email": "jan.kowalski@example.org",
  "city": "Warszawa",
  "birth_date": "1980-12-12"
}

”

Błędy walidacji

Kod aktywacyjny nie został podany:

Status: 422 Unprocessable Entity
{
  "message": "activation_token_is_blank"
}

”

Kod aktywacyjny jest nieprawidłowy:

Status: 422 Unprocessable Entity
{
  "message": "activation_token_not_found"
}

”

Kod aktywacyjny jest już zużyty:

Status: 422 Unprocessable Entity
{
  "message": "activation_token_already_used"
}

”
